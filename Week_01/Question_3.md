# Question 3: Identify front-end and back-end technologies used in web development and give examples of each.


Web development splits into Frontend (user-facing) and Backend (server-side logic/data), with common Frontend tools being HTML, CSS, JavaScript, React, Angular, and Vue, while Backend uses languages like Python, Java, PHP, Node.js, with frameworks like Django, Spring, and databases such as MySQL or MongoDB to handle data and server operations, ensuring the application works seamlessly.

### Front-End Technologies (Client-Side)
These technologies build the user interface (UI) and user experience (UX) that users directly see and interact with in their browser.

#### Languages:
    - **HTML (HyperText Markup Language):**  Structures the content of web pages (headings, paragraphs, images).
    - **CSS (Cascading Style Sheets):**  Styles the appearance (colors, fonts, layout).
    - **JavaScript (JS):**  Adds interactivity and dynamic features.

#### Frameworks and Libraries:
    -  **React:** A popular JavaScript library for building user interfaces (Facebook).
    -  **Angular:** A comprehensive framework for complex applications (Google).
    -  **Vue.js:** A progressive framework known for its ease of use.
    -  **Bootstrap/Tailwind CSS:** CSS frameworks for responsive design.

### Back-End Technologies (Server-Side)
These handle the server, application logic, databases, and APIs that power the frontend.

#### Languages:
    -  **Python:** Versatile, used with frameworks like Django (Instagram, Reddit).
    -  **Java:** Robust for enterprise-level applications (Spring Boot).
    -  **PHP:** Widely used for web development (WordPress, Laravel).
    -  **Node.js:** Allows JavaScript on the server, great for real-time apps.
    -  **Ruby:** Known for rapid development (Ruby on Rails).

#### Frameworks:
    -  **Django (Python):** For complex, database-driven applications.
    -  **Spring (Java):** A comprehensive framework for enterprise Java.
    -  **Laravel (PHP):** Elegant syntax for web artisans.
    -  **Express.js (Node.js):** Minimalist framework for building APIs.

#### Databases:
    -  **SQL (MySQL, PostgreSQL):** Relational databases for structured data.
    -  **NoSQL (MongoDB, Redis):** Document/key-value stores for flexible data.

**APIs (Application Programming Interfaces):** Allow frontend and backend to communicate.
